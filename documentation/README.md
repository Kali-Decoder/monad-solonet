# Monad Solonet

Run a complete Monad blockchain network locally — fully containerized, reproducible, and under your control.

---

## 🚀 Overview

**Monad Solonet** is a local execution environment that allows developers to spin up a full Monad network (validators + nodes) on their machine.

Instead of connecting to public networks, Solonet lets you:
- Start from **genesis**
- Control **validators and state**
- Test complex systems in isolation

---

## ✨ Features

- 🧱 Full local blockchain network (not just a node)
- 🔁 Reproducible Docker-based setup
- ⚡ Instant startup with prebuilt images
- 🧪 Multi-validator simulation
- 💰 Unlimited tokens (no faucet needed)
- 🛠️ Built-in tooling (forge, cast, staking-cli)

---

## 🖥️ Requirements

- Linux (x86_64) OR macOS (with VM)
- Docker installed
- Minimum:
  - 4 CPU
  - 16GB RAM
